5  投资组合构建与有效前沿曲线

5.1 目的与说明

  1. 有效前沿分析：
    • 使用均值-方差框架进行投资组合优化
    • 蒙特卡洛模拟生成随机权重组合

5.2 1. 等权重投资组合

5.2.1 1.1 组合构建

等权重（Equal Weight / 1/N）组合是最简单的投资组合策略，每只股票配置相同权重：

\[w_i = \frac{1}{N} = \frac{1}{5} = 20\%\]

优点： - 简单易行，无需复杂优化 - 自动实现分散化 - 不依赖历史数据的估计

  4. 夏普比率更高: 0.2345 vs 0.1889，风险调整后收益更优

5.3 2. 组合Beta可加性验证

5.3.1 2.1 CAPM理论预测

根据CAPM理论，组合的Beta等于各股票Beta的加权平均：

\[\beta_{portfolio} = \sum_{i=1}^{N} w_i \cdot \beta_i\]

对于等权重组合： \[\beta_{portfolio} = \frac{1}{N}\sum_{i=1}^{N} \beta_i = \frac{0.18 + 0.22 + 0.71 + 0.79 + 1.20}{5} = 0.621\]

5.4 3. 有效前沿分析

5.4.1 3.1 均值-方差框架

Markowitz（1952）提出的均值-方差模型：

  • 目标函数: 最大化夏普比率 \[ \max_{w} \frac{E(R_p) - R_f}{\sigma_p} \]

  • 约束条件: 权重和为1，权重非负

5.4.3 3.3 配置特征解读

最小方差组合（Minimum Variance Portfolio）: - 银行股(ICBC)占比最高达61.32% - 少量配置ZTE(1.38%)获取收益增强 - 中国石化(SINOPEC)占比28.04% - 几乎完全排除YILI和YUNNAN - 波动率9.36%，为所有组合最低

最大夏普组合（Maximum Sharpe Ratio Portfolio）: - 极度集中于ICBC(91.83%) - 少量配置ZTE(8.17%)提供收益来源 - 完全排除YILI、YUNNAN、SINOPEC - 夏普比率0.40，为最优风险收益比 - 年化收益6.63%，为所有组合最高

Tip

投资启示:

  1. 银行股的多重价值: 银行股在A股市场不仅Beta低、波动小，还能提供优异的风险收益比

  2. 科技股需谨慎: 虽然ZTE的预期收益高，但高波动性使其在风险调整后收益欠佳

  3. 消费医药的尴尬: 伊利和云南白药在有效前沿上几乎不被选中，说明其风险收益特征不够优秀

# 3. 蒙特卡洛模拟
n_simulations = 50000
np.random.seed(42)

sim_returns = []
sim_volatilities = []
sim_sharpes = []

for _ in range(n_simulations):
    # 随机生成权重
    weights = np.random.random(n_assets)
    weights = weights / np.sum(weights)  # 归一化
    
    # 计算组合指标
    ret = portfolio_return(weights, mean_returns)
    vol = portfolio_volatility(weights, cov_matrix)
    sharpe = (ret - RF_ANNUAL) / vol
    
    sim_returns.append(ret)
    sim_volatilities.append(vol)
    sim_sharpes.append(sharpe)

sim_returns = np.array(sim_returns)
sim_volatilities = np.array(sim_volatilities)
sim_sharpes = np.array(sim_sharpes)

# 3. 蒙特卡洛模拟 (使用狄利克雷分布)
n_simulations_dirichlet = 50000

dirichlet_returns = []
dirichlet_volatilities = []
dirichlet_sharpes = []

print(f'开始狄利克雷分布蒙特卡洛模拟 ({n_simulations_dirichlet} 次)...')

for _ in range(n_simulations_dirichlet):
    # 【核心修改】使用狄利克雷分布生成均匀的随机权重，直接满足和为1且均大于0
    # np.ones(n_assets) 表示在单纯形上进行完全均匀的采样
    # 将 alpha 参数设置为小于 1 的值（比如 0.15），迫使权重向极端值集中
    # alpha_value = 0.15 
    # weights_dirichlet = np.random.dirichlet(np.ones(n_assets) * alpha_value)
    weights_dirichlet = np.random.dirichlet(np.ones(n_assets))

    ret_d = portfolio_return(weights_dirichlet, mean_returns)
    vol_d = portfolio_volatility(weights_dirichlet, cov_matrix)

    dirichlet_returns.append(ret_d)

    dirichlet_sharpes.append(sharpe_d)
