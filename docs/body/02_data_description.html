3.1 股票选取

3.1.1 选取原则

本研究选取5只A股代表性股票，覆盖银行、消费、科技、医药、能源五大行业，力求体现A股市场的行业多样性：

序号 股票代码 股票名称 所属行业 选取理由
1 601398.SH 工商银行 银行 中国最大商业银行，国有银行代表
2 600887.SH 伊利股份 食品饮料 消费龙头，白马股代表
3 000063.SZ 中兴通讯 通信/科技 科技成长股，5G龙头
4 000538.SZ 云南白药 医药生物 医药防御型，品牌中药代表
5 600028.SH 中国石化 石油化工 能源周期型，央企代表

3.1.2 行业特征简析

  • 银行业（ICBC）: 受到央行货币政策和监管政策的强烈影响，利率是其核心驱动因素
  • 食品饮料（伊利）: 受到消费升级和原材料价格影响，具有防御性特征
  • 科技通信（中兴）: 受到技术迭代和政策影响，波动性较高
  • 医药生物（云南白药）: 人口老龄化和医疗改革推动，需求稳定
  • 石油化工（中石化）: 受国际油价和宏观经济周期影响

市场基准: 沪深300指数 (000300.SH) — 反映A股市场整体表现

3.2.2 对数收益率计算

本研究采用对数收益率计算股票收益：

\[r_t = \ln\left(\frac{P_t}{P_{t-1}}\right)\]

对数收益率的优点： - 时间可加性：多期收益率为各期收益率之和 - 正态分布近似：相比简单收益率更接近正态分布 - 数学性质优良：便于统计建模

3.2.3 无风险利率设定

日频无风险利率 0.025/252 ≈ 0.000099 用于计算日超额收益

Warning

重要发现: 所有资产收益率均显著偏离正态分布！

  • 所有资产的偏度均大于0，呈右偏（正偏）分布
  • 所有资产的峰度均大于3（正态分布峰度为3），存在显著的”尖峰厚尾”特征
  • Jarque-Bera检验的p值均为0，强烈拒绝正态性假设

投资启示: 正态分布假设下的VaR等风险指标可能低估实际风险！

3.3.3 波动率特征解读

  • 科技股（ZTE）: 年化波动率高达40.96%，风险最高
  • 银行股（ICBC）: 年化波动率仅10.34%，风险最低
  • 沪深300指数: 年化波动率19.00%，介于两者之间

3.4 数据获取代码示例

import akshare as ak
import pandas as pd
import numpy as np

# 下载个股日度行情
df = ak.stock_zh_a_hist(
    symbol='601398',      # 工商银行
    period='daily',      # 日频
    start_date='20190101',
    end_date='20260325',
    adjust='hfq'         # 后复权
)

# 下载沪深300指数
index_data = ak.index_zh_a_hist(
    symbol='000300',
    period='daily',
    start_date='20190101',
    end_date='20260325'
)

# 计算对数收益率
df['return'] = np.log(df['收盘'] / df['收盘'].shift(1))
